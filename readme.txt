1、Node.JS是什么?
   既不是语言,也不是框架,它是一个平台

   Node.js 中的 JavaScript
     1) 没有BOM、DOM
     2) EcmaScript 基本的 JavaScript 语言部分
     3) 在Node中为JavaScript提供了一些服务器级别的API
        文件操作的能力
        http服务的能力

2、Node.JS中核心模块
   1) 获取机器信息os
      os.cpus();     获取机器的CPU信息
      os.totalmem(); 获取机器的内存
   2) 操作路径path
   3) 文件读写fs
   4) HTTP服务http
